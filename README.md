## Contribuindo
Para contribuir com o código, instale o pre-commit:
```
pip install dev-requirements.txt
```
```bash
pre-commit install
```

```bash
alias discipline="pytest -vvs && git add -i && git commit"
```

Sempre use o alias discipline para commitar.
```bash
discipline
```
